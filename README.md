**Haven** is a modern real estate web application developed as our capstone project. The platform is designed to simplify property searching and management through an intuitive and user-friendly interface. It features a secure login and registration system, allowing users to create and manage their accounts, Users can explore properties and connect with professionals through the *Find Agent* page, while administrators have full control of the system via a dedicated admin dashboard. Additionally, Haven integrates an AI-powered agent to assist users with inquiries, recommendations, and navigation throughout the platform. This project demonstrates our skills in full-stack development, database management, and user-centered design.
The **Login Page** of Haven allows registered users to securely access their accounts by entering their credentials. It is designed with a simple and user-friendly interface, ensuring a smooth and safe login experience.
